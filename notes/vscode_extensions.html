Python

Microsoft's extension extension for Python.

Linting, debugging (multi-threaded, remote), Intellisense, Jupyter Notebooks, code formatting, refactoring, unit tests, snippets, and more.

Markdown All in One

Tools to edit Markdown files in VSCode: auto preview, table of contents generation, shortcuts...

Python Docstring Generator

Automatically generates detailed docstrings for Python functions.

It detects parameter names, default values and parameter types thrugh type hints. It also includes support for args, kwargs, decorators and exceptions.

Docstring formats: Google (default), Numpy and Sphinx

Code Spell Checker

Spelling checker for source code.

English is the default supported language. Other dictionaries can be installed.

Draw.io Integration

Integrates an offline version of Draw.io into VSCode.

It can edit .drawio , .drawio.svg and .drawio.png files.

It includes several themes, basic symbols, UML symbols, Azure icons and more.
